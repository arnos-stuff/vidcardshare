A CLI to make ready to share one page apps for videos¶

This CLI produces websites which you can serve from any bucket online, and the code aims to reproduce things like these. This is a very rough first attempt, nothing major yet, but it'll get better with time 😊

A Youtube download function baked in¶

If your video link is from www.youtube.com, this uses pytube to fetch the video automatically and then packages it with your website as a static dependency.
